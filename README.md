# AI Budget Planner – Goal-Oriented Financial Advisor

A full-stack web application that helps users plan realistic monthly savings
based on income, expenses, and personal financial goals using Machine Learning
and a Flask-based backend API.

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend API:** Python, Flask (REST API)
- **Machine Learning:** scikit-learn, Pandas
- **Others:** NumPy, Flask-CORS

## Key Features
- Goal-based budget planning
- ML-powered savings prediction
- Flask REST API for frontend–backend communication
- Real-time financial recommendations
- Clean and responsive user interface

## How to Run
1. Train the model  
   `python ai_financial_advisor.py train`

2. Start Flask API server  
   `python ai_financial_advisor.py`

3. Open `index.html` in browser
